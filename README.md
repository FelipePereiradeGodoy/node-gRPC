## gRPC + NestJS
Projeto com foco em implementar os conceitos de gRPC, tanto a parte do cliente como a servidor.

## Instalação
```bash
$ npm install
```

## Para rodar o App

Iniciar o cliente
```bash
$ npm run start:dev client-side
```

Iniciar o servidor
```bash
$ npm run start:dev server-side
```
